This is a backend coursework project built with Node.js and Express.

- Add, edit, delete apartments
- Upload images for apartments
- View all available apartments
- Session-based authentication

## Technologies Used
- Node.js
- Express.js
- MongoDB & Mongoose
- EJS (server-side rendering)
- Express-session
- Express-validator
- Multer (file uploads)
